# 使用指南说明

本文档说明如何维护和更新应用内的使用指南。

## 概述

使用指南位于 `src/components/HelpCenter.vue`，是一个集成在应用内的帮助中心。指南内容以文档数组的形式存储，支持以下特性：

- 文章浏览（快速开始、使用手册、使用技巧、更新说明）
- 全文搜索
- 图片展示
- 提示框
- 目录导航

## 文档结构

每个文档包含以下字段：

```javascript
{
  id: 'unique-id',           // 唯一标识符
  category: '分类名称',       // 显示在导航中的分类
  title: '文档标题',          // 显示在文章头部
  summary: '简短描述',        // 显示在导航项下方
  icon: IconComponent,        // Lucide 图标组件
  sections: [                 // 文章内容章节
    {
      title: '章节标题',
      paragraphs: ['段落内容'], // 可选
      image: {                // 可选
        src: '/screenshots/image.png',
        alt: '图片描述',
        caption: '图片说明文字'
      },
      items: ['列表项'],      // 可选，无序列表
      steps: ['步骤'],        // 可选，有序列表
      tip: '提示内容'         // 可选，提示框
    }
  ]
}
```

## 添加新文档

1. 在 `documents` 数组中添加新对象
2. 选择合适的分类和图标
3. 编写清晰的章节内容
4. 添加图片引用（如需要）

## 添加截图

### 1. 截图准备

运行应用并准备演示数据：

```bash
# 启动应用
npm run dev

# 生成演示数据
node scripts/seed-demo-data.mjs
```

### 2. 截图步骤

1. 使用 Windows 截图工具（Win+Shift+S）
2. 选择"矩形截图"模式
3. 截取应用界面
4. 保存到 `public/screenshots/` 目录
5. 使用描述性文件名（如 `today-view.png`）

### 3. 推荐截图尺寸

| 截图类型 | 推荐宽度 | 说明 |
|----------|----------|------|
| 全屏视图 | 1280px | 今日、计划、重要等主要视图 |
| 面板截图 | 480px | 设置、详情、编辑器 |
| 侧边栏 | 320px | 清单列表 |
| 小组件 | 400px | 子任务、标签 |

### 4. 更新图片引用

在 `HelpCenter.vue` 中更新图片路径：

```javascript
image: {
  src: '/screenshots/your-screenshot.png',  // 更新为实际文件名
  alt: '图片描述',
  caption: '图片说明文字'
}
```

## 内容编写指南

### 标题

- 使用简洁明了的标题
- 避免过长的标题
- 使用动词开头（如"创建任务"、"设置提醒"）

### 段落

- 每个段落聚焦一个要点
- 使用简单易懂的语言
- 避免技术术语

### 列表

- 使用列表展示多个要点
- 保持列表项简洁
- 使用一致的格式

### 提示框

- 用于补充重要信息
- 避免过度使用
- 内容简短有力

## 搜索功能

指南支持全文搜索，搜索范围包括：

- 文档标题
- 章节标题
- 段落内容
- 列表项
- 提示内容

搜索不区分大小写，实时更新结果。

## 样式定制

指南样式位于 `src/assets/styles.scss`，主要类名：

- `.help-center` - 主容器
- `.help-center__nav` - 导航栏
- `.help-center__article` - 文章区域
- `.help-center__section` - 章节
- `.help-center__figure` - 图片容器
- `.help-center__tip` - 提示框

## 常见问题

### Q: 如何添加新的文档分类？

A: 在 `documents` 数组中使用新的 `category` 值，文章会显示该分类名称。

### Q: 如何修改图片尺寸？

A: 截图会在阅读区内按自然比例缩放；需要重构图时，优先裁剪图片内容而不是在文章数据中设置固定宽度。

### Q: 如何添加代码块？

A: 目前不支持代码块，可以使用列表或段落展示代码相关内容。

### Q: 如何测试搜索功能？

A: 在应用中打开使用指南，在搜索框中输入关键词测试。
