# 界面设计规范

## 布局
- 桌面端采用三栏结构：左侧导航、中央任务列表、右侧任务详情。
- 左侧导航宽度约 286px，可折叠为 56px 图标栏（Rail）。
- 折叠态 Rail 顶部固定显示搜索图标，随后以分隔线区分主要视图图标（今日/收集箱/计划/重要）、清单入口（点击弹出浮动卡片）、维护视图图标（已完成/垃圾桶）、设置和展开按钮；当前视图用左侧彩色竖线标识。
- 搜索入口使用现有搜索视图和查询范围；点击 Rail 图标或按 `Ctrl/Cmd + K` 后聚焦搜索输入框，设置等模态层打开时不响应快捷键。
- 搜索视图支持列表和分组两种模式：列表模式连续展示结果；分组模式按所属清单归类，组头显示清单颜色、名称和结果数并支持折叠。搜索仅匹配标题、标签、重复规则及富文本备注的可见纯文本；结果展示任务分组与命中位置，并高亮标题和标签中的关键词。
- 清单浮动卡片：从图标右侧弹出的紧凑卡片（232px 宽，最大 400px 高），包含搜索框、置顶快捷区、分组清单列表和新建按钮，支持滑入动画。
- 中央列表为主工作区，包含视图标题、快速添加、今日建议、任务列表和空状态。
- 右侧详情宽度约 430px，显示任务属性、子任务、富文本备注和低频操作；支持拖拽调整宽度。
- 窗口较窄时详情面板覆盖在右侧；手机宽度隐藏侧栏，保留主列表和抽屉式详情。

## 视觉
- 使用 token 化主题，当前支持青绿、海蓝、紫罗兰、石墨四套配色。
- 背景以浅灰白为主，强调色只用于选中、焦点、重要动作和状态反馈。
- 多分组任务视图使用中性任务画布；分组颜色只作为标题左侧色条、图标、计数和轻微悬停底色的识别锚点，不使用整块彩色背景或任务行彩色边框。
- 控件圆角以 6px、8px、12px、16px 分层，避免大面积圆润卡片化。
- 结构性图标使用 `lucide-vue-next` 线性图标，不使用 emoji 作为导航或系统图标。
- 所有可点击控件必须有 hover、focus-visible、disabled 或 active 状态。

## 任务列表
- 每行包含完成框、标题、清单、日期、今日、重复、子任务、附件、标签等元信息。
- 鼠标悬停或键盘聚焦时显示行内操作：加入今日、重要、更多。
- 已完成任务默认可折叠，设置中可切换。
- 计划视图按日期分组，不在该视图直接添加任务。
- 今日视图显示建议加入今日的任务，点击建议即加入今日。

## 任务详情
- 顶部使用任务核心卡片：第一行放完成状态和快捷图标操作，包含加入今日、重要、置顶和关闭详情。
- 标题在概览卡中独占一行编辑；短标题保持一行紧凑，长标题最多两行，超出后在标题框内滚动。
- 概览卡只保留任务标题和轻量子任务进度，不重复展示清单、日期、提醒等属性信息。
- 顶部快捷图标使用低对比按钮组承载，避免图标散落；激活状态用青绿色强调，关闭操作保持弱化。
- 属性区使用轻量面板承载清单、截止日期、提醒、重复、优先级和标签；属性值区域必须支持长文本，长清单名最多两行显示且不得挤压下拉箭头。
- 子任务与富文本备注同时存在，不再使用详情/检查事项模式切换。
- 空白备注默认显示紧凑“添加备注”入口，聚焦后展开；工具栏以图标式块类型菜单代替原生下拉，并提供放大按钮进入独立编辑器。`/` 命令在光标旁以高层浮层提供标题、列表、待办、引用、分割线和图片插入，须始终避开视口边缘且不被详情面板遮挡。
- 备注编辑器支持标题、加粗、斜体、删除线、链接、列表、待办块、引用、分割线和图片。
- 详情面板内部滚动条使用窄型弱化样式，默认不抢视觉焦点，悬停时才提高可见度。

## 设置
- 设置从左上角进入，以右侧抽屉展示。
- 外观：4 套主题配色（青绿/海蓝/紫罗兰/石墨）、信息密度（舒适/紧凑）、侧栏折叠和详情面板默认显示。
- 任务显示：已完成任务在列表和分组中的显示、归位与默认展开规则。
- 偏好与提醒：启动视图、系统提醒通知、提醒声音、操作音效总开关和分类开关（任务/清单/分组）。
- 数据与维护：本地保存状态、隐私说明、垃圾桶保留天数和附件清理。
- 关于：应用图标、名称、版本和平台信息。
- 设置变更即时生效并通过本地数据持久化。
- 详情面板支持拖拽调整宽度（320-800px），宽度持久化到设置。

## 操作音效
- 基于 Web Audio API 合成，无外部音频文件，所有音效短促（60-140ms）、低音量，不干扰用户。
- 任务类使用 sine 波（明亮清脆），清单/分组类使用 triangle 波（柔和温暖），拖拽类使用 triangle 波（低沉不抢注意力）。
- 任务完成：三音琶音上行，明亮"叮铃"感；子任务完成：轻快双音上行；子任务取消完成：柔和下行双音"撤销"感。
- 任务添加：明亮双音上行；删除：三音阶下行；恢复：三音琶音上行。
- 清单置顶：明亮双音切换感；重命名：柔和单音确认感。
- 数据保存成功：极轻柔单音；保存失败：低沉双音下行警告。
- 支持总开关和分类独立开关（任务/清单/分组），设置持久化到 SQLite。

## 可访问性
- 图标按钮必须提供 `aria-label` 或可见文字。
- 焦点环不能移除。
- 颜色不能作为唯一状态提示，重要状态配合图标或文字。
- 交互区域最小尺寸接近 36px，桌面紧凑场景下仍需可点击。
- 动画遵守 `prefers-reduced-motion`。
