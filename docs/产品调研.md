# 产品调研

## 调研对象

| 产品 | 核心定位 | 主要能力 | 可借鉴点 |
| --- | --- | --- | --- |
| 滴答清单 / TickTick | 全平台任务、日历和效率工具 | 任务、日历、四象限、番茄、习惯、提醒、重复、标签、优先级、清单/文件夹、附件、协作、多视图 | 三栏桌面布局、任务详情面板、智能日期、任务属性密度 |
| Todoist | 项目化任务管理 | 自然语言添加、Today/Upcoming、项目、标签、过滤器、列表/看板/日历布局 | 快速输入、过滤器、任务组织模型 |
| Microsoft To Do | 轻量个人待办 | My Day、清单、提醒、截止日期、步骤、重复、分类、附件、Outlook 集成 | 每日计划、低门槛列表体验 |
| Things | 个人 GTD 任务管理 | Inbox、Today、Upcoming、Anytime、Someday、Areas、Projects、Tags、Checklist、Deadline | 克制视觉、个人任务层级、Today/Upcoming 模式 |
| OmniFocus | 高阶 GTD | Projects、Tags、Forecast、Review、Perspectives | 透视视图、复盘机制、复杂过滤 |
| Any.do | 任务和日历计划 | 任务、日历、每日计划、提醒、子任务、标签、看板、AI 助手 | 任务与日历结合、每日计划体验 |

## 滴答清单关键能力
- 快速添加任务，支持日期、提醒、重复、优先级、标签和所属清单。
- 桌面端典型信息架构为窄图标栏、清单侧栏、任务列表、任务详情面板。
- 任务可包含描述、检查事项、附件、评论、链接、子任务等扩展信息。
- 常用视图包括今天、最近 7 天、收集箱、已完成、垃圾桶，以及日历、看板、时间线等高级视图。
- 效率模块包括番茄专注、习惯打卡、四象限和日历。

## 当前产品范围
- 产品以本地个人任务管理为核心，并提供本机个人资料：用户可维护昵称和头像；未登录时资料仅存在当前设备。
- 必须具备：清单、任务增删改查、完成、回收站、日期提醒字段、优先级、标签、搜索、详情编辑、持久化、个人资料。
- 后续演进：账号绑定、云同步、局域网协作、完整日历；身份入口、数据模型和本地存储需避免阻碍这些能力。
- 当前不排期：番茄、习惯、四象限。
