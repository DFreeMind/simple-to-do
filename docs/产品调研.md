# 产品调研

## 调研结论

主流待办产品已经覆盖任务、日历、习惯、番茄、协作和跨设备同步。易简清单不以“功能更全”为竞争方式，而以**本地优先、桌面上的复杂事项推进、克制且可恢复的数据整理**为取舍标准。

调研结论用于判断方向，不等于功能承诺；当前已交付能力以 [产品需求文档](产品需求文档.md) 为准。

## 参考产品与可借鉴点

| 产品 | 代表能力 | 易简清单借鉴什么 | 不直接复制什么 |
| --- | --- | --- | --- |
| 滴答清单 / TickTick | 任务属性、三栏桌面布局、智能日期 | 信息密度、详情检查器、计划视图 | 习惯、番茄、完整日历、协作体系 |
| Todoist | 快速输入、Today / Upcoming、项目和标签 | 快速记录与日期解析 | 以过滤器和协作为中心的云服务模型 |
| Microsoft To Do | My Day、清单、提醒和步骤 | 每日聚焦与低门槛任务流 | Microsoft 账户与 Outlook 依赖 |
| Things | Inbox、Today、Upcoming、Areas | 克制的个人 GTD 心智模型 | 多层级项目体系 |
| OmniFocus | 复盘、透视视图 | “复杂事项需要可回顾”的判断 | 高学习成本的配置系统 |

## 产品取舍

### 必须持续做好

- 本机即可完成记录、安排、拆分、提醒、完成和恢复。
- 今日、计划、收集箱和任务详情之间的切换足够快，任务属性不会遮蔽下一步。
- 删除、附件清理和数据恢复有可理解的回退路径。
- Windows 与 macOS 都提供贴合桌面的窗口、通知和安装体验。

### 需要证明确有价值才做

- 周视图、轻量月历、统计、导出、批量整理和标签管理。
- 这些能力只有在帮助用户拆清或持续推进复杂事项时才进入路线图，不能因为竞品具备就加入。

### 当前不纳入

- 账号、云同步、共享清单、团队协作、评论与权限。
- 番茄、习惯、四象限和泛化项目管理。
