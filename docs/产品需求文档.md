# 产品需求文档

## 产品定位
「易简清单」是一款本地优先的个人任务与轻量笔记清单。产品以稳定、可日用的任务管理为基础，并为用户身份、局域网协作和云数据同步预留连续的演进路径；当前重点是收集、今日聚焦、计划、清单分组、轻量月历、个人复盘统计、富文本备注、本地保存和个人资料。

## 设计原则
- 打开应用默认进入「今日」，让用户马上知道今天要推进什么。
- 临时想法先进入「收集箱」，之后再安排日期、清单、标签或重要性。
- 所有可见按钮必须有真实行为，不展示不可用的同步、协作、账号等入口。
- 侧栏顶部是“个人空间入口”，承载头像、昵称、当前空间和数据状态；它不是“今日”的替代入口，点击今日仍始终进入今日视图。
- 用户即使不登录也拥有本机个人资料；本地资料不应被误导性地表述为云端账号或已同步身份。
- 任务详情不使用模式切换；采用“查看即编辑”的检查器形态，属性、子任务和富文本备注放在一个连续面板中，字段聚焦后进入明确编辑态。
- UI 保持现代桌面工具感：清爽、紧凑、低干扰、可切换配色。

## 核心视图
- 今日：每日聚焦视图，包含已加入今日或截止日期为今天的未完成任务，并显示建议加入今日的任务。
- 收集箱：默认入口，用于快速记录未整理任务。
- 计划：按截止日期聚合为已逾期、今天、明天、本周、以后。
- 重要：集中显示重要、高优先级或置顶任务。
- 月历：按月展示有截止日期的任务，支持点击日期新建任务、拖拽任务调整日期。
- 统计：展示未完成、逾期、本周到期、今日完成、7 天完成趋势和清单分布。
- 我的清单：用分组管理多个清单，支持新建、重命名、删除分组和清单；在侧栏中作为唯一滚动区。
- 个人空间：侧栏顶部显示可更换头像、昵称和“个人空间”状态；点击后进入空间中心。空间中心固定组织为身份资料、空间管理、连接与协作、数据与安全：空间管理承载本机保存状态、垃圾桶保留期和附件清理；后续在同一入口增加账户登录、同步状态和设备、空间切换、成员权限、备份恢复与数据保护，不重构侧栏信息架构。
- 搜索：按标题、备注、标签、重复规则检索任务。
- 已完成：回顾和清理完成任务。
- 垃圾桶：恢复或永久删除任务和清单。

## 核心功能
- 快速添加任务，支持基础自然语言解析：今天、明天、后天、下周、时间、每天、每周、每月、重要、`#标签`。
- 清单分组支持拖拽调整分组顺序、清单顺序和清单所属分组。
- 具体清单支持可组合筛选：完成状态（全部、未完成、已完成）、日期（全部、逾期、今天到期、未来到期、未设日期）和优先级（全部、高、中、低）。逾期仅包含截止日期已过且未完成的任务，但不改写用户已选择的其他筛选条件。筛选结果沿用当前列表或分组视图，切换清单或系统视图时恢复默认条件。
- 任务属性：加入今日、置顶、重要、所属清单、截止日期、提醒、重复、优先级、标签；已完成任务和子任务在元信息中显示完成时间及从创建到完成的用时。
- 截止日期选择器中的时间可不设置，或一键选择 09:00、14:00、18:00、20:00；自定义时间直接在同一行以原生时间输入控件编辑，支持键盘输入、系统数字时间选择和 ±15 分钟微调，失焦或按回车后写入任务。
- 提醒：设置截止日期时，默认在截止时间发送系统通知；用户可改为提前提醒或明确关闭单个任务的提醒。关闭窗口最小化到托盘时仍保持提醒。应用运行期间统一调度下一条提醒；从休眠恢复或重启后，对尚未投递的到期提醒补发一次，并避免重复提醒。
- 普通任务视图支持拖拽任务行调整当前视图顺序；按名称、日期、优先级等显式排序时拖拽临时禁用。
- 子任务：新增、编辑、完成、删除、拖拽排序。完成主任务时，所有未完成子任务同步完成；子任务全部完成不会自动完成主任务，主任务须由用户手动完成。
- 任务分组：清单支持分组视图和列表视图切换；分组模式下可创建任务分组（TaskGroup），支持图标、强调色、折叠；任务可归属分组，支持跨分组拖拽。分组强调色仅用于标题标识、图标、计数与交互反馈，任务画布保持中性以保障多分组阅读。已完成任务默认保留在原分组底部，分组头显示“已完成 / 总数”，并可在各分组头临时显示或隐藏。
- 富文本备注：空白时以紧凑入口展示，聚焦后展开编辑；支持常用格式工具栏、链接、`/` 命令、标题、列表、待办块、引用、分割线、图片粘贴或上传，并可在独立放大编辑器中完成长篇编辑。
- 设置：采用“外观、任务显示、偏好与提醒、关于”四类组织；数据维护统一在个人空间的“空间管理”中处理。支持配色、信息密度、侧栏和详情面板显示。配色中可选择是否让三栏背景使用当前主题的低饱和搭配色，默认保持白色界面；开启后，侧栏快捷入口和清单节点使用白色抬升层表达悬停和选中，详情摘要卡片使用同主题的低饱和搭配色。
- 个人资料：用户可修改昵称和头像。昵称去除首尾空白后必须为 1–24 个字符；头像可从本机选择图片、恢复默认头像。默认头像由昵称首字母/首个可见字符生成，不要求登录。自定义头像按本地内容寻址策略存储，不写入数据库 BLOB。
- 设置：可统一配置已完成任务在列表和分组中的显示规则，以及是否显示完成用时；可配置启动页、垃圾桶数据保留天数、系统提醒通知、提醒声音和操作音效。通知与音效关闭时保留子项偏好，重新开启后恢复生效。
- 帮助中心：侧栏底部和“设置 → 关于”均可打开；以“使用指南”呈现，使用指南针图标而非通用问号。帮助中心提供快速开始、使用手册、数据与维护、更新说明，支持按标题和正文搜索、文章目录跳转及小屏阅读。
- 本地自动保存，保存失败时展示可见错误。
- 图片附件按类型和年月分层保存，文件名使用内容 hash，避免同名冲突和单目录过大。

## 当前未纳入
- 账号登录、云同步、多人协作、共享清单与空间成员管理；个人资料仅代表当前设备上的本地身份。
- 系统级完整日历、外部日历同步、番茄、习惯、四象限。
- 评论通知和团队工作流。
- 复杂统计报表、导出和多维筛选。
- 普通文件附件暂不开放，当前仅开放图片附件；目录和数据结构为后续普通附件预留。

## 验收标准
- 首屏显示今日视图，空状态、快速添加和详情面板正常。
- 每个可见按钮都能完成对应操作或给出明确反馈。
- SQLite 数据能稳定保存并恢复任务、清单、设置和垃圾桶状态。
- 添加图片附件后重启应用，图片仍可预览，主数据文件不写入外置附件的 base64 内容。
- 删除清单后清单和其中任务进入垃圾桶；恢复清单后任务回到原清单。
- 设置修改能持久化，重启后保持。
- 昵称和头像修改后，侧栏个人空间入口立即更新；重启后保持。恢复默认头像后不再引用原头像文件。
- 使用指南可从侧栏和“关于”打开，任一文章只展示一个阅读视图；支持搜索、切换文章、文章内目录跳转和 Esc/遮罩关闭。
- `npm run frontend:build` 和 `npm run tauri:build` 可通过，若 Tauri 构建受本机环境限制需明确说明原因。
