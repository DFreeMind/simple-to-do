# 技术架构

## 架构目标

易简清单是 Tauri 桌面应用，而不是独立 Web 产品。架构优先保证本地数据可靠、Windows/macOS 行为可控，以及在没有网络或账号时仍可完成全部核心任务流程。

```text
Vue 3 视图与交互
        │
Pinia 任务状态、派生视图、业务操作
        │
src/services/platform.js（唯一平台边界）
        │
Tauri invoke / 插件 ───────────── Web 开发回退仅用于调试
        │
Rust commands → SQLite、附件、恢复点、窗口、通知
```

## 分层职责

### 前端：Vue 3、Pinia、Vite、SCSS

- 组件负责界面、焦点、弹层和交互反馈；不直接读写本地文件或调用 Tauri 全局对象。
- `src/stores/task.js` 负责任务、清单、分组、个人资料、设置、时钟状态、搜索、排序、提醒调度和保存节流。
- `src/services/platform.js` 是唯一的平台封装层：调用 Rust command、读写附件、处理通知权限和非 Tauri 调试回退。
- 富文本使用 Tiptap；结构性图标使用 `lucide-vue-next`；操作音效由 `src/utils/sound.js` 使用 Web Audio API 合成。

### 原生层：Tauri 2 与 Rust

- `load_data` / `save_data`：在前端状态对象与 SQLite 表之间转换，并在写入时剥离附件预览 URL。
- 图片与头像：选择、导入、读取、富文本图片修复和未引用头像清理。
- 数据维护：SQLite 结构检查、迁移备份、本机恢复点、附件扫描、清理站恢复与永久清理。
- 平台行为：窗口关闭策略、托盘/Dock 恢复、系统通知；Windows 任务提醒点击可恢复窗口并定位任务。
- 原生层不承担产品业务规则；业务规则仍由 Pinia 管理，避免桌面 API 与任务规则耦合。

## 数据与兼容

- 主数据位于 Tauri `app_data_dir()` 下的 `simpletodo.db`；附件、头像、恢复点与数据库处于同一应用数据目录。
- SQLite 的 `databaseSchemaVersion` 与前端状态 `schemaVersion` 分开维护。结构变更由 Rust 事务处理；前端状态缺字段由迁移器与 normalize 补齐。
- 数据库读取、健康检查、备份或迁移失败时，应用进入保护状态并阻断写入，不能以默认空数据覆盖现有数据。
- 单独运行 Vite 浏览器页时可使用 localStorage 调试回退，但它不是产品数据路径，也不能代替 Tauri 能力。
- 详细字段见 [数据模型](数据模型.md)，表与文件布局见 [数据库设计](数据库设计.md)。

## 通知、音效与窗口行为

- 提醒由前端维护下一次待投递任务；应用启动、恢复或运行中会检查尚未投递的到期提醒。
- Windows 使用可回调的原生 toast 处理任务通知点击；macOS 使用通知中心能力。通知权限和系统设置会影响最终展示。
- 操作音效使用 Web Audio 的 oscillator、滤波和包络合成，避免 macOS WKWebView 对 OGG/Vorbis 音频的兼容问题；只为有结果的操作播放。
- 关闭窗口可设置为隐藏到后台或直接退出。隐藏后，Windows 可通过通知区域托盘恢复；macOS 可通过 Dock 或菜单栏恢复。两端应验证等价用户结果，而不是依赖相同 API。

## 发布与更新

- 版本以 `package.json`、`src-tauri/tauri.conf.json` 和 `src-tauri/Cargo.toml` 为准，三者必须一致。
- GitHub Actions 的 `release.yml` 手动构建 Windows NSIS 包、创建草稿 Release 并生成签名更新资产；发布前由 `scripts/release-preflight.mjs` 校验版本和资源命名。
- macOS 安装包按 Release 规范独立构建并追加到同一个 Release。Release 资产是用户唯一可信的安装来源。
- 更新器使用 GitHub Release 的 `latest.json` 与内置公钥验证。某个 Release 没有签名更新清单时，应用必须提示用户到 Release 页面手动下载，而不是宣称在线更新可用。

## 关键模块

| 位置 | 责任 |
| --- | --- |
| `src/App.vue` | 应用壳、主题、加载与窗口级事件 |
| `src/components/Sidebar.vue` | 系统视图、清单导航、窗口内快捷键与个人空间入口 |
| `src/components/TaskList.vue` | 快速添加、筛选、排序、搜索与任务分组视图 |
| `src/components/TaskDetail.vue` | 任务属性、子任务、富文本备注和附件 |
| `src/components/SettingsPanel.vue` | 外观、行为、通知反馈、更新说明 |
| `src/components/ProfilePanel.vue` | 本机资料、空间维护与恢复点 |
| `src/components/HelpCenter.vue` | 应用内帮助的唯一内容源与截图入口 |
| `src/components/ClockWorkspace.vue` | 专注模式选择、当前会话、任务关联和当天摘要 |
| `src-tauri/src/main.rs` | SQLite、文件、恢复、窗口与原生通知 commands |

## 架构边界

不维护 Electron 运行时或独立 Web 版本；不为未上线的账号、同步、协作建立假接口。若未来加入这些能力，必须保持本地 SQLite 可读写、个人资料不依赖登录、数据恢复可用。
