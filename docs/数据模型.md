# 数据模型

## Task
- `id: string`
- `title: string`
- `description: string`
- `descriptionHtml: string`
- `editorMode: "detail" | "checklist"`
- `completed: boolean`
- `completedAt: string | null`
- `deleted: boolean`
- `deletedAt: string | null`
- `pinned: boolean`
- `listId: string`
- `dueDate: string | null`
- `reminderAt: string | null`
- `repeatRule: string | null`
- `priority: 0 | 1 | 2 | 3`
- `tags: string[]`
- `subtasks: Subtask[]`
- `comments: Comment[]`
- `attachments: Attachment[]`
- `createdAt: string`
- `updatedAt: string`

## List
- `id: string`
- `name: string`
- `icon: string`
- `isSystem: boolean`

## Subtask
- `id: string`
- `title: string`
- `completed: boolean`

## Comment
- `id: string`
- `text: string`
- `author: string`
- `createdAt: string`

## Attachment
- `id: string`
- `path: string`
- `url: string`
- `createdAt: string`

## 后续预留
- `Tag`：统一管理颜色、名称和排序。
- `Reminder`：支持多提醒和相对提醒。
- `Habit`：习惯名称、频率、打卡记录。
- `FocusSession`：番茄开始时间、结束时间、关联任务。
