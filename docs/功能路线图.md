# 功能路线图

## 阶段 1：Tauri 迁移和本地核心任务
- 初始化 Tauri 2 运行时。
- 用 Rust command 替换 Electron 的数据读写、图片选择、图片读取。
- 修复任务计数、回收站、复制、清除已完成等基础问题。
- 完成滴答清单风格主界面和搜索。

## 阶段 2：任务组织增强
- 清单文件夹。
- 标签管理视图。
- 自定义过滤器。
- 更完整的重复规则。
- 任务批量操作。

## 阶段 3：日历与计划
- 月/周/日历视图。
- 拖拽调整日期。
- Today/Upcoming 计划流。
- 与系统日历导入导出预留。

## 阶段 4：效率模块
- 四象限视图。
- 番茄专注和专注记录。
- 习惯打卡和统计。

## 阶段 5：同步与协作
- 账号体系。
- 云同步。
- 共享清单和评论通知。
- 多端冲突处理。
