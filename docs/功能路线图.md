# 功能路线图

## 阶段 1：现代本地个人清单
- 三栏现代 UI：导航、任务列表、统一详情。✅
- 今日聚焦：加入今日、今日建议、今日快速添加。✅
- 基础视图：收集箱、计划、重要、搜索、已完成、垃圾桶。✅
- 清单分组：分组和清单的新建、重命名、删除、移动、拖拽排序。✅
- 清单回收站：删除清单先进入垃圾桶，可恢复或永久删除。✅
- 任务分组：清单内任务分组管理，支持分组视图/列表视图切换、分组图标、颜色、折叠。✅
- 轻量月历：按月查看任务、点击日期新建任务、拖拽任务调整日期。（已实现组件，未接入侧栏入口）
- 个人复盘统计：关键数量、7 天完成趋势、清单分布。（已实现组件，未接入侧栏入口）
- 任务详情：属性、子任务、富文本备注和备注内图片。✅
- 设置：配色、信息密度、侧栏折叠、启动页、详情面板、已完成分组。✅
- 操作音效：基于 Web Audio API，每类操作独特音色，分类独立开关。✅
- 系统托盘：关闭窗口最小化到托盘，右键菜单显示/退出。✅
- 本地自动保存、旧数据兼容和分层图片附件存储。✅
- SQLite 主存储，表结构见 `docs/数据库设计.md`。✅

## 阶段 1 验收
- 所有可见按钮都有真实行为。
- `npm run frontend:build` 通过。
- 涉及 Tauri 时运行 `npm run tauri:build`，或记录无法运行的本机原因。
- 完成稳定功能后使用中文 commit 提交。

## 阶段 2：个人空间与整理效率增强
- 个人资料：本地昵称、可更换头像、默认首字母头像和恢复默认头像。✅
- 侧栏顶部升级为个人空间入口：展示头像、昵称、个人空间及本地数据状态；Rail 保留头像按钮；今日仍是独立默认视图。✅
- 标签管理视图。
- 自定义过滤器。
- 重复规则完善。
- 批量选择和批量移动/删除/完成。
- 快速添加自然语言解析增强。

## 阶段 3：计划体验增强
- 轻量周视图。
- 拖拽调整日期。
- 过期任务处理流。
- 与系统日历导入导出预留。

## 阶段 4：连接与协作（完成本地资料和数据边界验证后）
- 账号绑定与登录状态：将本地个人资料与可选云端身份关联，不以登录作为使用应用的前提。
- 云同步：保持本地 SQLite 为可用数据副本，提供同步状态、冲突提示、手动恢复与设备管理；入口位于个人空间的“连接与协作”。
- 局域网协作：共享空间、成员与权限、清单共享和变更同步；入口位于个人空间的“连接与协作”。
- 数据与安全：在个人空间提供备份策略、恢复点、数据迁移、设备管理和本地数据保护。
