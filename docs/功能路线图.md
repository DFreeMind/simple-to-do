# 功能路线图

## 阶段 1：可日用本地清单
- 固定「易简清单」产品壳、窗口标题和基础图标。
- 清理未实现入口，只保留本地个人任务管理可用功能。
- 修复任务计数、视图新建、回收站、复制、移动、清除已完成等基础问题。
- 打磨滴答清单风格主界面、搜索、任务详情、日期、优先级、标签、子任务、评论和附件。
- 强化本地 JSON 兼容、保存失败反馈和 Tauri 平台封装。

## 阶段 1 验收
- 首次启动、重启恢复、旧 JSON 兼容都正常。
- 今天、最近 7 天、收集箱、自定义清单、搜索、已完成、垃圾桶视图行为一致。
- `npm run build` 通过；涉及 Tauri/Rust 变更时 `npm run tauri:build` 通过或记录阻塞原因。
- 每个稳定功能验证后使用中文 commit 提交。

## 阶段 2：任务组织增强
- 清单文件夹。
- 标签管理视图。
- 自定义过滤器。
- 更完整的重复规则。
- 任务批量操作。

## 阶段 3：日历与计划
- 月/周/日历视图。
- 拖拽调整日期。
- Today/Upcoming 计划流。
- 与系统日历导入导出预留。

## 阶段 4：效率模块
- 四象限视图。
- 番茄专注和专注记录。
- 习惯打卡和统计。

## 阶段 5：同步与协作
- 账号体系。
- 云同步。
- 共享清单和评论通知。
- 多端冲突处理。
