# 安装与使用

## 易简清单适合谁

易简清单是一个本地优先的个人桌面待办应用，适合把一件复杂事情拆成下一步、在“收集—安排—完成”之间持续推进的人。它提供清单、子任务、计划、提醒、富文本备注和本机恢复点；不要求注册账号，也不提供云同步或多人协作。

## 下载与安装

从 [GitHub Releases](https://github.com/DFreeMind/simple-to-do/releases/latest) 下载最新安装包。请以 Release 页面实际展示的资产为准：

- Windows：下载 `simple-to-do_<version>_x64-setup.exe`，双击安装。升级前如应用正在运行，请先退出应用，避免安装器无法替换文件。
- macOS Intel：下载 `simple-to-do_<version>_x64.dmg`，打开后将“易简清单”拖入“应用程序”文件夹。首次打开如被系统拦截，请在 Finder 中按住 Control 点击应用并选择“打开”。

当前公开 Release 已提供 Windows x64 和 macOS Intel 包；其他架构是否可用以 Release 资产为准。

## 第一次使用

1. 在**收集箱**记录临时想法，或在**今日**直接添加马上要做的事。
2. 给任务补充清单、截止日期、提醒、优先级和标签；复杂事项继续拆成子任务。
3. 在**计划**中查看逾期、今天、明天、本周和以后；不再合适的任务可重新安排到今天或明天。
4. 在任务详情中写备注、插入图片，或将任务放入清单内的任务分组。
5. 删除会先进入垃圾桶。进行大批量整理前，可在“个人空间 → 数据与安全”创建恢复点。

## 常用快捷键

| 操作 | Windows | macOS |
| --- | --- | --- |
| 打开搜索 | `Ctrl + K` | `⌘ + K` |
| 快速添加任务 | `N` | `N` |
| 将选中任务加入今日 | `D` | `D` |
| 关闭当前弹层 | `Esc` | `Esc` |
| 富文本加粗 / 斜体 | `Ctrl + B` / `Ctrl + I` | `⌘ + B` / `⌘ + I` |

快捷键只在应用窗口处于前台时生效。输入框、富文本编辑器和打开的弹层会优先接收按键，避免误操作。

## 数据、隐私与恢复

- 任务、设置和本机个人资料保存在应用数据目录内的 SQLite 数据库；图片附件以独立文件保存，数据库不保存图片 base64。
- 应用不要求账号，不上传任务内容，也不提供云同步。系统提醒需要操作系统授予通知权限。
- 可以在“个人空间 → 空间管理”扫描未引用附件，将其移入清理站、恢复或永久删除。
- 可以在“个人空间 → 数据与安全”创建、打开、恢复或删除本机恢复点。恢复前应用会先创建安全点；恢复点包含数据库、附件和头像。

如需迁移或重装，先创建恢复点，再妥善保存该恢复点目录。不要直接编辑 SQLite 数据库或附件目录。

## 已知边界

- 不支持账号登录、云同步、共享清单、协作、外部日历同步、番茄钟、习惯和四象限。
- 普通文件附件尚未开放；当前支持图片附件。
- 自动更新仅在对应 Release 提供已签名更新清单时可用；没有清单时，应用会提示前往 Release 手动下载安装包。

## 反馈问题

请在 [GitHub Issues](https://github.com/DFreeMind/simple-to-do/issues) 中说明：系统版本、应用版本、发生步骤、预期结果和实际结果。请勿附带真实任务内容、数据库或其他隐私文件；可使用脱敏截图复现问题。
